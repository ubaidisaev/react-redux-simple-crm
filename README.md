## Simple Tasks and Clients management app  -  [Demo](https://ubaid-react-redux-simple-crm.web.app)

Using technology:

- React
- TypeScript
- Redux
- Sass

### Installation

```sh
$ git clone https://github.com/ubaidisaev/react-redux-simple-crm.git
$ cd react-redux-simple-crm
$ yarn i
$ yarn start
```
